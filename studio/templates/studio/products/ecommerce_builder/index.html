    What technical skills do I need to use this?

    Basic familiarity with Django is helpful but not required. The documentation covers everything from setup to deployment. If you can follow instructions and use a terminal, you can get this running. For customisation, some Python/Django knowledge is beneficial.
    {# FAQ 2 #}

    What hosting do I need?

    Any VPS that can run Python/Django works. We recommend Digital Ocean, Hetzner, or Railway. A basic £5-10/month VPS is plenty for most stores. The documentation includes deployment guides for popular hosting platforms.
    {# FAQ 3 #}

    Can I use this for physical products too?

    This system is optimised for digital products with instant delivery. While you could adapt it for physical products, you'd need to add shipping logic yourself. For physical products, a dedicated solution like WooCommerce might be better suited.
    {# FAQ 4 #}

    Do you offer support?

    The package includes comprehensive documentation. For additional help, you can reach out via email for guidance. This is indie software — I'm one developer, not a support team — but I do my best to help customers succeed.

    Can I see a demo before buying?
